# 🔎 DataLens

DataLens is an open-source intelligent dataset visualization engine.

Upload a JSON or CSV file and DataLens:
- Profiles your schema
- Detects column types (date, numeric, categorical)
- Recommends the appropriate chart automatically
- Enables dynamic filtering and slicing
- Supports large datasets interactively

## Tech Stack
- Python
- Streamlit
- Plotly
- Pandas

## Run Locally

pip install -r requirements.txt
streamlit run app.py
